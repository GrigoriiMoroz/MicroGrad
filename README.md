### Micrograd

Репозиторий содержит реализацию micrograd (механизм forward / backward pass), а так же базовые реализации нейрона, слоя нейронов и MLP.

Структура проекта:

	•	micrograd/engine.py — реализация вычислительного графа и автоматического дифференцирования
	•	micrograd/nn.py — базовые строительные блоки нейронных сетей (нейроны, слои, MLP)
	•	micrograd/__init__.py — инициализация модуля
	•	test/ — тесты для проверки корректности реализации backprop
	•	Demo_one.ipynb, Demo_two.ipynb — демонстрационные ноутбуки с примерами использования
	•	MicroGrad_WorkBook.ipynb — учебный ноутбук с пошаговыми экспериментами и пояснениями
	•	images/ — вспомогательные изображения для ноутбуков
